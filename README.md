# 校园外卖
## 项目简介
本项目参考了黑马程序员的《苍穹外卖》项目，是面向高校师生打造的轻量化校园生活服务平台，采用前后端分离架构开发，聚焦校园内商品/餐食的线上浏览、下单、订单管理等核心场景，解决校园内线下消费效率低、信息不透明等问题，为师生提供便捷的线上消费体验，同时为商家提供标准化的商品、订单管理能力。
项目为个人独立开发完成，覆盖需求分析、数据库设计、后端接口开发、前后端联调、系统测试全流程。

## 技术架构
### 后端技术栈
- 核心框架：SpringBoot
- 持久层框架：MyBatis + MyBatis-Plus
- 数据库：MySQL 8.0
- 缓存中间件：Redis 6.x
- 权限认证：JWT
- 项目构建：Maven
- 开发工具：IntelliJ IDEA

### 前端技术栈
- 前端框架：Vue3
- UI组件库：Element Plus
- 网络请求：Axios

## 核心功能
1.  **用户认证模块**：基于JWT实现用户登录、权限校验，保障接口访问安全
2.  **商品管理模块**：实现商品分类、商品信息增删改查、上下架管理，支持多维度商品搜索
3.  **购物车模块**：实现购物车商品添加、修改数量、删除、批量选择等核心功能
4.  **订单管理模块**：实现订单创建、订单状态流转、订单查询、历史订单查看等功能
5.  **店铺管理模块**：实现店铺营业状态设置、基础信息管理功能
6.  **缓存优化**：基于Redis实现高频访问的商品分类数据缓存，减少数据库压力，提升接口响应速度
7.  **定时任务**：基于Spring Task实现超时未支付订单自动关闭、库存自动回滚功能
